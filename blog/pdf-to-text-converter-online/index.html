Frequently Asked Questions

Can I extract text from scanned PDFs?

Our tool extracts text that's already in the PDF. For scanned PDFs (which are images), you'll need OCR (Optical Character Recognition) software first. We extract digital text, not text from images.

What's the maximum file size?

Currently 50MB, which covers 99% of PDFs. Most text documents are under 5MB. Large files are usually due to embedded images, which don't affect text extraction.

Why is some text missing or garbled?

This usually happens with:

  • PDFs using unusual fonts not embedded in the file
  • Password-protected PDFs (unlock first)
  • Corrupted PDF files
  • Image-based PDFs (scanned documents)

Can I convert multiple PDFs at once?

Currently one at a time for quality control. For bulk conversion, extract each PDF and combine the text files using our other tools.

Do you support other formats like Word or Excel?

We specialize in text manipulation. For now, it's PDF to text only, but you can use our CSV tools for spreadsheet data.
